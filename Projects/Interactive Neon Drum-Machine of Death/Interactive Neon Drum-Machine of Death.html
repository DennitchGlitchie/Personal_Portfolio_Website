Interactive Neon: Drum-Machine of Death

A completed version of this project was submitted to Hackaday’s Musical Instrument Creation Contest. Documentation can be seen here. Also, this project was featured on arduino’s blog.

Summary:

Working with artists in the LA (@dani_bonnet) to implement advanced light control on relevant Neon Art Signs. Tubes can be programmed to light up with MIDI protocol such as MIDI notes or MIDI clock from a DJ application.

Technical Skill Utilized:

Microcontroller programing for High Power Application Circuit design for audio -> visual conversion system Artistic collaboration with visionaries
